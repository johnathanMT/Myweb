# Explore — cinematic hover scenes (zero-config, optional)

Drop a still image named after the interest (slug) and it auto-binds to that
card's hover preview. Slug = the interest title, lowercased, spaces/symbols → `-`.

```
cosmology.webp          quantum-theory.webp     linguistics.webp
fe-it-study.webp        occultism.webp          fortune-prediction.webp
general-knowledge.webp  unconditional-joke.webp
```

- Any of `.webp` / `.jpg` / `.png` / `.avif`. Vite bundles + content-hashes them.
- If a scene is missing, the card falls back to a themed cinematic gradient
  (never a broken image).
- Use imagery you have the rights to (your own stills / licensed / royalty-free).
- Tip: ~1200px wide, compressed, landscape framing.
